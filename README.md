# Fractals SIMD

## Overview

Library containing following set of fractals implemented using SIMD:

| Fractal      | SSE3 | AVX2 |
|--------------|------|------|
| Mandelbrot   | X    | X    |
| Burning ship | X    | X    |
| Julia set    | X    | X    |
