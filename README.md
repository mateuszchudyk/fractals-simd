## Overview

Library containing following set of fractals implemented using SIMD:

|              | SSE3 | AVX2 |
|--------------|------|------|
| Mandelbrot   | +    | +    |
| Burning ship | +    | +    |
| Julia set    | +    | +    |

**Built-in colormaps:**

red, green, blue, yellow, cyan, magenta, gray, hot, jet, cividis, twilight, turbo
